# React Parcel Template

Create React App Using Parcel Bundler !
If you want to write React at once, you find it.

## Usage

```
$ git clone `https://github.com/realdennis/${thisRepo}`
$ cd ${thisRepo}
$ yarn install
$ yarn dev
or
$ yarn build // For production
```
